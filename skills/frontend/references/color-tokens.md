# Color Token Dictionary

This file defines the active CSS color tokens to be applied for any frontend components generated by this skill.

## Base Variables

```css
:root {
  /* Brand Colors */
  --color-primary: #0F172A;
  --color-primary-hover: #1E293B;
  
  /* Background & Surface */
  --bg-default: #FFFFFF;
  --bg-surface: #F8FAFC;
  
  /* Text */
  --text-primary: #020617;
  --text-secondary: #475569;
  
  /* Semantic States */
  --color-success: #10B981;
  --color-warning: #F59E0B;
  --color-error: #EF4444;
}
```

Never use hardcoded hex values in component outputs. Always map to these standard tokens.
