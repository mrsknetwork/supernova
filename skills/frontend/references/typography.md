# Typography Standards

This file defines the typography guidelines for the frontend skill.

## Font Families
- **Primary:** 'Inter', sans-serif
- **Monospace:** 'JetBrains Mono', monospace

## Type Scale
- **H1:** 2.5rem, line-height: 1.2, font-weight: 700
- **H2:** 2rem, line-height: 1.25, font-weight: 600
- **H3:** 1.75rem, line-height: 1.3, font-weight: 600
- **Body:** 1rem, line-height: 1.5, font-weight: 400
- **Small:** 0.875rem, line-height: 1.4, font-weight: 400
